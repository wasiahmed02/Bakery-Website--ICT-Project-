# Bakery Website (ICT-Project)
**Project Website:** [Visit Bakers](https://wasiahmed02.github.io/Bakery-Website--ICT-Project-/)
# Project Description
This project is a basic web development project using HTML, CSS, and JavaScript. The goal was to create a responsive and interactive bakery website that demonstrates practical web development skills learned during the semester.  

**Key Features:**
- Multi-page website: Home, Menu, About, Login, and Cart  
- Responsive design for desktop and mobile screens  
- Interactive elements using JavaScript (e.g., navigation, image handling)  
- Organized content: hero section, services, story, team, and deals  
- Clean styling using CSS including flexbox, cards, and typography  

**Learning Outcomes:**
- Applying HTML and CSS to create structured and styled web pages  
- Using JavaScript for basic interactivity  
- Understanding layout, design principles, and responsive web development
